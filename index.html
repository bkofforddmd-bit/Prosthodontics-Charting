AI settings — your Anthropic API key

Each person using this app enters their own key. It is stored only in this browser on this device (localStorage) and is used to bill AI features (plan generation, documentation, radiograph assist) to your own Anthropic account. It is never shared with anyone else or saved into patient files.

Get a key at console.anthropic.com → API Keys. Your usage and billing are managed there.
